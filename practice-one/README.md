# GETTING STARTED

* Clone the repository

```bash
git clone https://github.com/TienNguyenDinh/training-html-css.git
```

* Move to folder `practice-one`

```bash
cd training-html-css/practice-one
```

* Checkout to branch `practice-one`

```bash
git checkout practice-one
```

* Install packages

```bash
npm install
```

* Run the server

```bash
npm start
```
